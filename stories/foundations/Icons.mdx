# Icons

The design system uses [Lucide](https://lucide.dev) for iconography. Icons are imported directly from `lucide-react`.

## Usage

```tsx
import { Check, ChevronRight } from 'lucide-react';

<Check className="h-4 w-4" />
```

## Icon Set

<IconGallery>
